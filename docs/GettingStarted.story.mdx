# Getting Started

## Installing

You can install reaselct with [NPM](https://www.npmjs.com/package/reaselct) or Yarn.

- NPM: `npm install reaselct --save`
- YARN: `yarn add reaselct`

## Configuration
reaselct uses [CSS Variables](https://developer.mozilla.org/en-US/docs/Web/CSS/Using_CSS_variables)
to configure the colors and styles of the library. Below are the variables
that we use.

You should include these variables in your CSS file or
on a container element such as the body like:

```html
<style>
  body {
    --color-select-input: white;
    --color-on-select-input: rgb(51, 51, 51);
    --color-select-input-placeholder: rgb(145, 145, 145);
    --color-select-input-border: rgb(204, 204, 204);
    --color-select-input-disabled: rgb(145, 145, 145);

    --color-select-input-error: rgba(255, 0, 0, 0.5);
    --color-select-input-icon: rgb(145, 145, 145);

    --color-select-chip: rgb(230, 230, 230);
    --color-select-chip-icon: rgb(145, 145, 145);
    --color-select-chip-border: transparent;
    --color-on-select-chip: rgb(51, 51, 51);

    --color-select-menu-border: rgb(204, 204, 204);
    --color-select-menu: white;
    --color-select-menu-group: grey;
    --color-on-select-menu-item: rgb(51, 51, 51);

    --color-select-menu-item-active: #DEEBFF;
    --color-on-select-menu-item-active: rgb(51, 51, 51);

    --color-select-menu-item-selected: #2684FF;
    --color-on-select-menu-item-selected: white;
  }
</style>
```

You can make the select use a dark theme by changing the variables to:

```html
<style>
  body {
    --color-select-input: #202124;
    --color-on-select-input: #bdc1c6;
    --color-select-input-placeholder: rgb(145, 145, 145);
    --color-select-input-border: #3c4043;
    --color-select-input-disabled: rgb(145, 145, 145);

    --color-select-input-error: rgba(255, 0, 0, 0.5);
    --color-select-input-icon: #757b80;

    --color-select-chip: #303134;
    --color-select-chip-icon: #757b80;
    --color-select-chip-border: transparent;
    --color-on-select-chip: #bdc1c6;

    --color-select-menu-border: #3c4043;
    --color-select-menu: #202124;
    --color-select-menu-group: grey;
    --color-on-select-menu-item: #bdc1c6;

    --color-select-menu-item-active: #303134;
    --color-on-select-menu-item-active: #bdc1c6;

    --color-select-menu-item-selected: #2684FF;
    --color-on-select-menu-item-selected: #bdc1c6;
  }
</style>
```
